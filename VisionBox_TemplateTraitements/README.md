# Guide de développement des traitements VisionBox

Ce dossier contient un modèle complet pour créer un nouveau traitement VisionBox.

Fichiers fournis :
- monexempledetraitement.h
- monexempledetraitement.cpp
- adaptateur_monexempledetraitement.h
- adaptateur_monexempledetraitement.cpp

Modifiez les noms, paramètres et logique interne selon vos besoins.
Ajoutez ensuite dans traitement_factory.h le bloc de code fourni dans adaptateur_monexempledetraitement.h.
